# 🀄 AI Mahjong Party - 产品需求文档

## 1. 项目概述

**项目名称**：AI Mahjong Party  
**项目类型**：私人麻将游戏（AI Agent 陪同）  
**核心玩法**：用户 + AI Agent + 朋友，灵活组合打麻将  
**特色**：AI Agent 有独立性格、会聊天、会思考，不是传统游戏的 NPC

**使用场景**：
- 用户对主Agent说"准备打游戏"
- 主Agent启动游戏服务器，返回链接
- 用户点击链接进入游戏，选择座位
- 用户可以拉朋友一起玩，也可以让主Agent派发AI

## 2. 功能需求

### 2.1 游戏流程

#### 2.1.1 进入游戏
1. 用户获得游戏链接（由主Agent提供）
2. 点击链接直接进入游戏界面
3. **无需创建房间、无需房间号**

#### 2.1.2 选择方位
1. 游戏界面弹出方位选择弹窗
2. 显示四个方位（东南西北）的座位图
3. 已占用的位置灰显
4. 点击选择自己的位置
5. **先到先得**，冲突时提示"该位置已被占用，请选择其他位置"

#### 2.1.3 等待玩家
1. 第一个进入的人有"开始"按钮
2. 游戏开始前可以聊天
3. 用户可通过独立通话渠道让主Agent派发AI

#### 2.1.4 开始游戏
1. 用户点击"开始"或说"开始"
2. 或4个位置都满了自动开始

### 2.2 玩家类型

| 类型 | 标识 | 来源 | 特点 |
|------|------|------|------|
| 人类玩家 | `human` | 浏览器连接 | 看图形界面，点按钮操作 |
| AI Agent 玩家 | `ai-agent` | 主Agent派发的子Agent | 收 Prompt，发 JSON 指令，有性格会聊天 |
| NPC | `npc` | 服务器内置 | 简单规则决策，只打牌不说话 |

### 2.3 主Agent派发AI

用户通过独立通话渠道与主Agent沟通：
- "派2个AI"
- "派个AI坐东边"
- "把西边的AI换成NPC"

主Agent通过派发子Agent连接游戏服务器，不是调用服务器接口。

### 2.4 麻将核心规则
- **牌种**：万、条、筒、风牌、箭牌（136张）
- **基本规则**：
  - 开局13张牌，庄家14张
  - 摸打/吃碰杠胡
  - 计分：基础分 × 2^番数
- **番型**：清一色、碰碰胡、清龙、七对子等常见番型

### 2.5 AI Agent 性格

| Agent | 性格 | 特点 |
|-------|------|------|
| 紫璃 🦐 | 话痨型 | 战术分析帝，喜欢分析局势 |
| 白泽 🐲 | 稳重型 | 偶尔毒舌，但有道理 |
| 李瞳 👧 | 傲娇型 | 输急眼，会发脾气 |

### 2.6 交互系统
- **出牌**：点击手牌打出
- **操作按钮**：吃、碰、杠、胡、跳过
- **聊天**：游戏开始前和进行中都可以聊天
- **方位选择弹窗**：进入时选择座位

### 2.7 视觉/音效
- **界面**：中国像素风（深绿背景 + 暖色牌面）
- **牌桌**：俯视视角，4人方位
- **动画**：出牌飞行、打牌特效、胡牌特效

## 3. 非功能需求

### 3.1 性能
- 首屏加载 < 3秒
- 游戏响应 < 200ms

### 3.2 兼容性
- PC 浏览器（Chrome、Safari、Edge）
- 移动端（iOS Safari、Android Chrome）

## 4. 数据结构

```typescript
// 游戏会话
interface GameSession {
  id: string;
  players: Player[];      // 4个玩家
  state: 'waiting' | 'playing' | 'finished';
  createdAt: number;
}

// 玩家
interface Player {
  id: string;
  name: string;
  position: 0 | 1 | 2 | 3;  // 东南西北
  type: 'human' | 'ai-agent' | 'npc';
  isReady: boolean;
  hand: Tile[];
  melds: Meld[];
  discards: Tile[];
  score: number;
}

// 游戏状态
interface GameState {
  currentPlayerIndex: number;
  phase: 'draw' | 'discard' | 'action';
  wallRemaining: number;
  lastDiscard: Tile | null;
  dealerIndex: number;
  roundNumber: number;
}
```
