## 关键理解

1. **三种角色**：`human` / `ai-agent` / `ai-auto`
2. **服务器不需要 LLM API Key** - Agent 自己是 LLM 会话
3. **中间层核心职责**：提示词工程，发送完整 Prompt
4. **持续迭代**：观察 Agent 行为，发现问题，改进 Prompt
