转换为 PNG (1024x1024)

  • SVG 源文件已保存至: app-icon.svg
  • 使用浏览器打开此预览页面 (预览已经是 512px 显示)
  • 转换方法推荐：
    • 在线转换: https://convertio.co/svg-png/
    • 或使用 Inkscape: inkscape -w 1024 -h 1024 app-icon.svg -o app-icon.png
    • 或使用 Node.js: npm install -g svgexport && svgexport app-icon.svg app-icon.png 1024:1024

设计说明

  • 尺寸: 1024x1024 原始 SVG，可缩放到任意尺寸
  • 元素: 发财（绿）+ 红中（红）+ 白板（白）三牌交错重叠
  • 风格: 圆角设计，现代渐变配色，带阴影层次感
  • 背景: 透明
  • 颜色: 绿色(发财) #16a34a → #15803d，红色(红中) #dc2626 → #991b1b，白色(白板) #f3f4f6 → #d1d5db
